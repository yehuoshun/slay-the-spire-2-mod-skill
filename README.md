# STS2 Mod 开发技能

- 🚀 从零搭建项目（.csproj / manifest / 入口）

- 🃏 自定义卡牌（ShunCard 基类 + 链式配置）
- 🎭 自定义修正器（SyncedModifierModel + 状态管理）
- ✨ UI 组件开发（Overlay 挂载 / Canvas 绘制 / 捕获主题字体）
- 👥 联机适配（Host/Client 检测 + 交互锁定）
- 🌍 本地化（JSON 文件 + LocString）
- 🔧 实用模式（AppliedTracker / ConditionalWeakTable / 反射+事件触发）
- ⚠️ 常见坑点
- 📦 发布流程
- 🔗 API 速查表
