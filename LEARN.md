# 学习流程

当老板说"学习"或发来教程链接时，按以下流程执行：

---

## 学习步骤

1. **读内容** — 读老板发的链接/文档/代码
2. **提炼关键知识** — 提取可复用的代码模板、API 用法、常见坑
3. **写 references** — 创建 `references/<模块名>.md`，结构：
   - 概述
   - 代码模板（标注引用来源）
   - API 速查
   - 常见问题
4. **更新 SKILL.md** — 参考资料表加新行
5. **更新 README.md** — 状态列表勾选 + 鸣谢追加
6. **commit + push**

---
